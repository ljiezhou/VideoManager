《Video Cleaner》核心功能 UI 原型图（V1）

🎨 UI 配色方案与风格搭配
•	主色调 (Primary Color): #3D8AFF （清爽蓝色，强调按钮、选择状态）
•	辅助色 (Accent Color): #FF6B6B （红色，用于删除、警告提示）
•	背景色 (Background Color): #F9FAFB （淡灰白，主页面背景）
•	卡片/列表项背景色: #FFFFFF （白色卡片，略有阴影提升层次）
•	字体颜色:
•	主标题/高亮文字: #1F1F1F （深灰，保证可读性）
•	副标题/说明文字: #6B6B6B （中灰，信息提示）
•	链接/操作按钮文字: #3D8AFF （与主色调一致）
•	进度条颜色: #3D8AFF （压缩进度条保持与主色统一）
•	圆角: 12dp （卡片、按钮统一圆角）
•	阴影: 2dp 阴影用于卡片浮层和压缩状态弹窗
